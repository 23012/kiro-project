# 💊 중앙대학교병원 약 처방 지원 시스템

## 📌 이 프로젝트는 무엇인가요?

중앙대학교병원 응급실·외래 진료 시 **환자 중심의 약 처방 의사결정을 지원**하는 시스템입니다.

환자의 과거 처방 이력을 조회하고, 약물 간 상호작용·다제약물 주의사항을 한눈에 확인할 수 있어
의료진이 **더 안전하고 빠른 처방 판단**을 내릴 수 있도록 돕습니다.

### 💡 사용 프로세스

1. 환자 정보(이름, 등록번호, Triage 등급)와 주증상이 상단에 표시됩니다
2. 기간·진료과 필터를 설정하고 **조회** 버튼을 클릭합니다
3. 과거 유사 증상 처방 이력이 내원일자별로 표시됩니다
4. 각 내원 이력 하단의 **⚠ 주의사항** 토글을 열면 약물 상호작용과 다제약물 주의사항을 확인할 수 있습니다

---

## ⚙️ 주요 기능

| 기능 | 설명 |
|------|------|
| 환자 정보 카드 | 이름, 성별/나이, 등록번호, Triage 등급, 내원 정보를 한눈에 표시 |
| 주증상 배너 | 진단코드(R42 등)와 동반 증상을 경고 배너로 강조 |
| 과거 이력 조회 | 기간(전체/6개월/1년/2년/3년/직접입력)과 진료과로 필터링 |
| 처방 이력 테이블 | 내원일자별 약품명, 용량/용법, 처방기간, ATC 계통, 상태 표시 |
| 주의사항 토글 | 각 내원 이력 하단에 약물 상호작용 + 다제약물 주의사항 표시 |
| 진료과 필터 | 응급의학과, 신경과 등 15개 진료과 필터링 지원 |
